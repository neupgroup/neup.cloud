# Create Startup Programs Page

I will create a new page at `/system/startup` to list all programs that automatically run when the Linux server restarts.

## Implementation Steps

### 1. Backend Logic (Server Action)
Create `src/app/system/startup/actions.ts` to fetch enabled systemd services.
- **Function**: `getStartupServices(serverId: string)`
- **Command**: `systemctl list-unit-files --type=service --state=enabled`
- **Output Parsing**: Parse the command output to extract service names and their states.

### 2. Frontend Components
Create `src/app/system/startup/startup-client.tsx` to display the data.
- **Features**:
  - Fetch and display the list of startup services.
  - Search/Filter functionality (by service name).
  - Status indicators (Enabled).
- **UI**: Use Shadcn UI `Card` and `Table` or list layout consistent with the "Processes" page.

Create `src/app/system/startup/page.tsx`:
- The main page wrapper that retrieves the selected server ID from cookies and renders the client component.

### 3. Sidebar Update
Update `src/app/layout.tsx`:
- Add a new "Startup" link to the **System** section of the sidebar.
- **Icon**: Use the `Play` or `Power` icon from `lucide-react`.
